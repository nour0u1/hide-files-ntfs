# hide-files-ntfs
A comprehensive, step-by-step laboratory demonstration showcasing how NTFS Alternate Data Streams (ADS) can be used to hide files on Windows operating systems. This project serves as an educational resource for cybersecurity students, IT professionals, and digital forensics enthusiasts.

Project Purpose
- **Educational**: Understand NTFS file system internals
- **Awareness**: Learn how malware can hide using legitimate system features
- **Defensive**: Train in detecting hidden streams for security purposes
